# 毕业设计

### 一个多模态的边缘计算于SBC的机器小狗

#### 大脑部分

* 摄像头和麦克风直接连接于SBC  
* 语音通过关键词识别，使用AISHELL-1，THCHS-30以及部分自采数据，微调DS-CNN获得指令，用麦克风阵列判断语音方向和人体匹配
* 视觉方面使用MediaPipe Hands判断方向实现方向判断，MediaPipe Pose找到人体
* 情绪方面使用YAMNet-TFLite和DS-CNN获得一个归一化的浮点数表示急迫程度
* 使用ros2的信息发送唤醒对应线程
* 最后使用规则连接（还没想好）实现模态融合与反应
     
#### 脊髓部分

* 大脑通过串口下命令
* 舵机模拟关节和肌肉
